Fundamentos de software

Todo desarrollador/ra debe potenciar primero sus habilidades básicas en conocimientos base requeridos antes de afrontar un desarrollo web y sus múltiples tecnologías.

HTML+CSS

El primer paso para desarrollar tus páginas y sitios web, es tener una base sólida en maquetación con HTML y darles los mejores estilos y diseños con CSS.

Repositorios

Toda persona que inicie en desarrollo de software debe manejar control de versiones y repositorios de sus proyectos y para eso existe GIT, GITHUB y GITLAB.

JavaScript

El lenguaje de programación líder para el desarrollo web, dominar JavaScript es dominar la web.
